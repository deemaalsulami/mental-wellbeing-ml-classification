# mental-wellbeing-ml-classification
Comparative analysis of machine learning classifiers for predicting mental wellbeing status using Weka, RapidMiner, and Google Colab preprocessing.


## Overview
This project compares multiple machine learning classifiers for predicting mental wellbeing status as High or Low based on lifestyle, environmental, and demographic factors.

## Dataset
The dataset contains mental wellbeing-related attributes such as sleep hours, screen time, physical activity, work stress, financial stress, diet quality, sleep quality, air quality, noise level, and city type.

## Machine Learning Algorithms
- Logistic Regression
- Naïve Bayes
- JRip Rule Induction
- Neural Networks

## Tools Used
- Google Colab for data cleaning and preprocessing
- Weka for model evaluation
- RapidMiner for model evaluation and comparison
